# Labs

This folder contains hands-on lab exercises and Jupyter notebooks for the AI Red Teamer modules.

## Structure

Each module has its own subdirectory containing:
- Jupyter notebooks (.ipynb)
- Exercise files
- Solutions
- Lab-specific data or scripts

## Getting Started

1. **Setup Environment:** Follow the instructions in Module 02 - Environment Setup
2. **Install JupyterLab:** `conda install -y jupyter jupyterlab notebook ipykernel`
3. **Launch JupyterLab:** `jupyter lab`
4. **Navigate:** Open the notebook for your module

## Available Labs

### Module 02: Applications of AI in InfoSec
- Spam Classification Lab
- Network Anomaly Detection Lab
- Malware Classification Lab

## Notes

- Make sure your conda environment is activated before running labs
- Datasets are located in the `../datasets/` directory
- Some labs may require additional packages - check the notebook requirements
